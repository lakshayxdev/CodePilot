# 🚀 CodePilot

CodePilot is a full-stack AI-powered code analysis platform that helps developers understand, debug, optimize, and improve their code with intelligent AI assistance. It provides multiple analysis tools through a clean, modern interface built for productivity.

---

## ✨ Features

- 🔐 JWT based User Authentication (Signup & Login)
- 💻 Monaco Code Editor
- 🤖 AI-Powered Code Analysis
- 📝 Code Explanation
- 🐞 Bug Detection & Debugging
- 📊 Time & Space Complexity Analysis
- ⭐ Code Quality Assessment
- 🧪 Test Case Generation
- 📜 Analysis History
- 🗑 Delete Individual Analysis
- 📱 Fully Responsive UI
- 🌙 Modern Dark Theme
- ⚡ Fast & Smooth User Experience
